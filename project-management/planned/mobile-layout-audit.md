# Mobile Layout Audit

**Priority:** medium

## Goal

Audit all app pages on mobile screen sizes and fix glaring layout issues. Target problems include tabs overflowing their content panels, header buttons consuming too much horizontal space, modals not centering correctly or exceeding available dimensions, and edge-positioned components overlapping each other.

## Notes

- Scope covers all existing pages, not just new ones.
- Fix only glaring/broken layouts — not a full mobile redesign.
- Examples of issues to address: tab overflow, oversized header buttons, misaligned or oversized modals, overlapping edge components.
