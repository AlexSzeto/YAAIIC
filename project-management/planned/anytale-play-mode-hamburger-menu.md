# AnyTale Play Mode Hamburger Menu

**Priority:** medium

## Goal

Replace the close/home button in AnyTale play mode's top-side UI with a hamburger menu. The menu should contain all top-side controls: dialog TTS settings, BGM settings, restart, and exit.

## Notes

- Currently there is a close/home button at the top of the play mode UI.
- The hamburger menu replaces that button and becomes the single entry point for all top-level controls.
- Controls to move into the menu: dialog TTS settings, BGM settings, restart, exit.
