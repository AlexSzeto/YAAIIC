# Ambient Brew Player

**Priority:** low

## Goal

Import the Ambient Brew player from an external project into this app, with a new layout visually styled after AnyTale play mode — mobile-friendly and simplified controls. The player pairs infinitely regenerating music (genre-based, ephemeral playlist) with infinitely looping background videos, where both the music prompts and video image prompts are generated by an LLM.

## Notes

- Source: import from the existing external Ambient Brew project rather than building from scratch.
- Inspired by the existing AnyTale editor music player but with the simplified, touch-friendly aesthetic of play mode.
- Controls should be minimal — play/pause, skip, genre selection.
- Music playlist is ephemeral (session-only, not persisted to media-data.json).
- Regeneration is infinite: as tracks finish or the queue runs low, new tracks are generated automatically.
- LLM generates the prompt for each music track to produce seamless looping audio.
- Background is a looping video driven by LLM-generated image prompts (not stored in media database).
- Open question: should genre be selected by the user upfront, inferred from context, or both?
