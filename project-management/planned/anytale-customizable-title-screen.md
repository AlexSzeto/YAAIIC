# AnyTale Customizable Title Screen

**Priority:** medium

## Goal

Use image generation to dynamically replace the AnyTale title screen. Generation can be triggered on a schedule or on-demand by the user, with the prompt configurable in the app config.

## Notes

- Schedule vs. on-demand should both be supported modes.
- The generation prompt lives in config so users can tune the style without touching code.
- Open question: where/how is the generated title image stored and served.
