# AnyTale Play Mode Loading Improvements

**Priority:** medium

## Goal

Improve the AnyTale play mode loading experience with a dedicated load screen between chapter transitions, progress indicators showing percentage of queued tasks complete, and the ability for the user to enter a chapter as soon as loading finishes. Additionally, suppress the built-in queue progress banners that appear on other pages while AnyTale is generating.

## Notes

- A dedicated loading screen should appear during chapter transitions, not just on initial load.
- Loading screens should display progress as a percentage of queued tasks completed.
- User should be able to enter the chapter immediately once loading is done (no auto-advance; explicit action).
- The global queue progress banners visible on other pages should be hidden or removed while AnyTale play mode is active.
